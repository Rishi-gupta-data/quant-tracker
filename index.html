Foundations — Math, Stats & Python Basics

Weeks 1–6

Build the mathematical and programming bedrock. By the end you should be comfortable with linear algebra, calculus, probability, and basic Python data manipulation.

1.1 Mathematics for Finance
Linear Algebra
3Blue1Brown — Essence of Linear Algebra Linear Algebra (Lay) — Free PDF Practice on Google Colab
Calculus & Optimization
3Blue1Brown — Essence of Calculus MIT OCW — Single Variable Calculus MIT OCW — Multivariable Calculus
Probability Theory
MIT — Probability & Statistics (18.650) Probability Course (free textbook)
1.2 Python Programming Primer
Python Basics → Intermediate
Mosh — Python for Beginners (6hr) Official Python Tutorial learnpython.org (interactive)
NumPy, Pandas, Matplotlib
NumPy Full Course — freeCodeCamp Pandas Official Tutorials Python Data Science Handbook (GitHub)
📌 Phase 1 Mini-Project

Download Nifty 50 daily OHLCV data from NSE/Yahoo Finance. Compute daily returns, rolling mean & std, and plot distributions using Matplotlib/Seaborn. Implement a covariance matrix from scratch using NumPy.

2

Quantitative Methods — Statistics & Econometrics

Weeks 7–14

Master the statistical and econometric tools that underpin every quant strategy: hypothesis testing, time-series analysis, regression, and Monte Carlo simulation.

2.1 Statistics for Finance
Descriptive & Inferential Statistics
CrashCourse Statistics (YouTube) Think Stats 2e (free PDF) Think Stats — Notebooks (GitHub)
Stochastic Processes
MIT OCW — Stochastic Processes Applied Stochastic Processes (PDF)
2.2 Financial Time-Series Analysis
ARIMA, GARCH & Volatility Models
Time Series Analysis — ritvikmath (YT) Tsay — Analysis of Financial Time Series statsmodels TSA Documentation
Cointegration & Pairs Trading
QuantPedia — Pairs Trading Overview Quantopian Research Notebooks (GitHub)
2.3 Monte Carlo Simulation
GBM, Option Pricing via MC
Monte Carlo for Finance — YouTube Tutorial Python for Finance (Hilpisch) — GitHub Notebooks
📌 Phase 2 Mini-Project

Fit ARIMA + GARCH models to Nifty 50 or S&P 500 returns. Forecast 30-day volatility. Implement a Geometric Brownian Motion simulator in Python and price a European call option via Monte Carlo.

3

Financial Markets & Derivatives

Weeks 15–20

Understand market microstructure, financial instruments, and derivative pricing models — the domain knowledge every algo trader must have.

3.1 Market Microstructure
Order Books, Bid-Ask, Market Making
Market Microstructure — NPTEL IIT Lectures Hasbrouck — Empirical Market Microstructure Limit Order Book Survey (arXiv)
3.2 Equity & Fixed Income
Equities, ETFs, Futures, FX Basics
Khan Academy — Core Finance CFA Institute — Free Reading (Level I samples) Zerodha Varsity (India-specific, excellent free)
3.3 Options & Derivatives
Black-Scholes, Greeks, Volatility Surfaces
MIT OCW — Financial Math (15.401) Hull — Options, Futures & Other Derivatives (get library copy) DX Analytics — Derivatives Pricing (Python) Zerodha Varsity — Option Theory (free)
📌 Phase 3 Mini-Project

Implement a Black-Scholes option pricer in Python. Build an implied volatility surface from live NSE/BSE option chain data. Replicate the payoff of a bull spread and iron condor strategy.

4

Python for Finance — Data & Backtesting

Weeks 21–26

Use Python's financial ecosystem to acquire data, engineer features, and rigorously backtest trading strategies.

4.1 Market Data & Feature Engineering
yfinance, NSEpy, CCXT for Crypto
yfinance Documentation NSEpy (Indian markets) ML for Trading — Jansen Notebooks (GitHub)
Technical Indicators — TA-Lib / Pandas-TA
Pandas-TA (130+ indicators, free) Python Technical Analysis — Part Time Larry (YT)
4.2 Backtesting Frameworks
Backtrader, Vectorbt, Zipline
Backtrader Docs (free, event-driven) VectorBT Docs (vectorized, fast) Backtrader Full Course — YouTube
Performance Metrics — Sharpe, Drawdown, Alpha
PyFolio — Performance Analytics (free) QuantPedia — Performance Metrics Guide
📌 Phase 4 Mini-Project

Build a complete backtest of a Moving Average Crossover + RSI strategy on Nifty 50 data using Backtrader. Compute Sharpe ratio, max drawdown, CAGR. Optimize parameters using a grid search and guard against overfitting with walk-forward analysis.

5

Machine Learning for Algorithmic Trading

Weeks 27–36

Apply the full ML pipeline — from supervised prediction to unsupervised regime detection — to real financial data, with emphasis on avoiding lookahead bias and overfitting.

5.1 Supervised Learning for Finance
Regression — Lasso, Ridge, ElasticNet
StatQuest — Regularized Regression (YT) Scikit-learn Linear Models Docs
Classification — SVM, Random Forests, XGBoost
StatQuest with Josh Starmer (full ML series) Aurélien Géron — Hands-On ML (get free on GitHub) Hands-On ML Notebooks (GitHub)
Walk-Forward Validation & Purged Cross-Validation
Lopez de Prado — Advances in Financial ML (SSRN preview) Advances in Financial ML — Lopez de Prado (key text) MlFinLab — Financial ML Library (GitHub)
5.2 Unsupervised Learning & Regime Detection
K-Means, GMM, HMM for Market Regimes
Hidden Markov Models — Victor Lavrenko (YT) hmmlearn Library (GitHub)
Portfolio Optimization — Mean-Variance, Risk Parity
PyPortfolioOpt Docs (free library) Portfolio Optimization in Python — YouTube
5.3 Alternative Data & Sentiment
NLP for News & Earnings, FinBERT
FinBERT Paper (arXiv) FinBERT Implementation (GitHub) FinBERT on Hugging Face (free inference)
📌 Phase 5 Mini-Project

Build an ML-driven long/short equity strategy: engineer 50+ features (price, volume, volatility, sentiment), train an XGBoost classifier with purged k-fold CV, run a full backtest, and compute feature importances. Target Sharpe > 1.0 on out-of-sample data.

6

Algorithmic Trading Systems

Weeks 37–44

Design and implement production-grade algo trading systems: strategy types, execution algorithms, risk management, order management, and live broker integration.

6.1 Strategy Design
Trend Following, Mean Reversion, Momentum
Ernie Chan — Quantitative Trading (highly recommended) QuantPedia — 800+ Strategy Descriptions (free tier) Two Centuries of Momentum (SSRN, free)
Statistical Arbitrage & Market Making
Avellaneda-Stoikov Market Making Model (PDF) Stat Arb & Pairs Trading — QuantLib (YT)
6.2 Execution Algorithms
TWAP, VWAP, Implementation Shortfall, Stealth Algos
Almgren-Chriss Optimal Execution (PDF) TWAP Explained — Investopedia
6.3 Risk Management
VaR, CVaR, Position Sizing, Kelly Criterion
VaR and CVaR Explained — YouTube Quantopian — Position Concentration Risk (GitHub) McNeil et al. — Quantitative Risk Management
6.4 Live Trading Integration (India + Global)
Zerodha Kite Connect API (India), Interactive Brokers API
Kite Connect API Docs (free sandbox) Interactive Brokers TWS API Docs PyKiteConnect Library (GitHub) Streak — No-code Algo (free tier, NSE/BSE) AmiBroker (backtesting + live, popular in India)
📌 Phase 6 Mini-Project

Build a complete event-driven algo trading system in Python: (1) real-time data handler, (2) strategy module (momentum + mean reversion), (3) portfolio risk manager with position limits and stop-loss, (4) order execution via paper trading on Zerodha sandbox or Alpaca (US paper trading). Simulate 30 days of paper trading.

7

AI & Deep Learning for Finance

Weeks 45–52

Apply deep learning — LSTMs, Transformers, Reinforcement Learning — to price forecasting, strategy generation, and autonomous trading agents.

7.1 Neural Networks Foundations
Feedforward Networks, Backprop, Regularization
DeepLearning.AI — Deep Learning Specialization (free audit) Goodfellow et al. — Deep Learning Book (free online) Fast.ai — Practical Deep Learning Notebooks (GitHub)
7.2 Recurrent Networks for Time Series
RNN, LSTM, GRU for Price Forecasting
LSTM Explained — Andrej Karpathy's blog + video ML for Trading — RNN Chapter Notebooks (GitHub) Attention is All You Need (Transformer paper, arXiv)
Temporal Fusion Transformer, N-BEATS for TS
Darts — Time Series Library (free, incl. TFT) TFT Example Notebook (GitHub)
7.3 Reinforcement Learning for Trading
Q-Learning, DQN, PPO for Portfolio Management
DeepMind RL Lectures (YouTube) FinRL — Deep RL for Finance (GitHub, active) FinRL Paper (arXiv)
7.4 Generative AI & LLMs in Finance
LLM Agents for Market Research, FinGPT
FinGPT — Open-Source Finance LLM (GitHub) FinGPT Paper (arXiv) Hugging Face — Financial LLMs Overview
📌 Phase 7 Mini-Project

Train an LSTM + attention model to forecast next-day Nifty returns. Build a FinRL DQN agent that learns a long/short trading policy on historical data. Compare agent performance vs. your Phase 5 XGBoost strategy by Sharpe ratio and maximum drawdown.

★

Capstone Projects (Portfolio-Ready)

Final 4 weeks

Complete 2–3 of these projects and publish to GitHub. They demonstrate job-ready skills to hedge funds, prop desks, and quant firms.

Capstone A — End-to-End Systematic Strategy
ML for Trading Reference (Jansen — GitHub)
  • Universe selection: Nifty 500 or S&P 500 constituent filtering
  • Feature engineering: 60+ alpha factors (price, vol, fundamental, sentiment)
  • ML model: LightGBM with purged walk-forward CV
  • Execution: Transaction cost model + slippage estimation
  • Risk: VaR limits, sector neutrality, max drawdown guardrails
  • Report: Monthly performance attribution and tearsheet (PyFolio)
Capstone B — High-Frequency / Intraday Algo
  • Tick data acquisition via Zerodha WebSocket or Alpaca streaming API
  • Implement VWAP execution algorithm with slippage tracking
  • Build an order book imbalance signal and test its predictive power
  • Deploy on local machine with real-time monitoring dashboard (Streamlit)
Capstone C — RL Trading Agent
  • Environment: Custom OpenAI Gym env wrapping historical Nifty data
  • Agent: PPO (Proximal Policy Optimization) via Stable Baselines3
  • State: Price history + technical indicators + portfolio state
  • Reward: Risk-adjusted return (Sharpe-based reward shaping)
  • Benchmark comparison vs. buy-and-hold and MA crossover strategy
